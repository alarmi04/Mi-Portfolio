## Ejecutalo en local

Clona el proyecto

```bash
  git clone https://github.com/alarmi04/Mi-Portfolio.git
```

Ves a la carptea del proyecto

```bash
  cd mi-portfolio
```

Instala las dependencias

```bash
  npm install
```

Inicia el servidor

```bash
  npm run dev
```
